Como usar: clique no cabeçalho de um grupo para o minimizar/abrir; clique numa linha para ver toda a informação do anúncio e escrever observações; use o selector de cada linha para marcar «A analisar», «A concorrer» ou «Descartado». As marcações e observações ficam guardadas na nuvem e acompanham-no em qualquer computador. Os contadores por cima das tabelas mostram quantos anúncios estão em cada marcação — clique num contador para ver só esses. Ao marcar como verificado (semáforo), é enviado automaticamente um email à outra pessoa a avisar que tem coisas para ver. Ao abrir, a página mostra os últimos 7 dias (abre mais depressa); para ver mais atrás use os campos «Ver de/até» ou o botão «Todos». A página atualiza-se sozinha todos os dias — pode carregar em «↻ Atualizar» a qualquer momento para ver as novidades.
Fonte: dados oficiais do Portal BASE / IMPIC (dados.gov.pt). O link «DR» de cada anúncio abre o PDF oficial no Diário da República.
