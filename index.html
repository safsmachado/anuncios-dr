Como usar: clique no cabeçalho de um grupo para o minimizar/abrir; clique numa linha para ver toda a informação do anúncio e escrever observações; use o selector de cada linha para marcar «A analisar», «A concorrer» ou «Descartado». As marcações e observações ficam guardadas neste computador automaticamente. Para ver datas anteriores use os campos «Ver de/até». A página atualiza-se sozinha todos os dias — pode carregar em «↻ Atualizar» a qualquer momento para ver as novidades.
Fonte: dados oficiais do Portal BASE / IMPIC (dados.gov.pt). O link «DR» de cada anúncio abre o PDF oficial no Diário da República.
